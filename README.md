# Zeah Mess Hall

A RuneLite plugin that guides you through making pineapple pizzas at the Hosidius Mess Hall by highlighting the next object or item to interact with based on your inventory.

## What it does

Highlights the next step in the pineapple pizza workflow:

| Inventory state | Highlight |
|---|---|
| Empty bowls | Sink |
| Bowls of water | Food cupboard → flour |
| Pizza bases | Food cupboard → tomato |
| Incomplete pizzas | Food cupboard → cheese |
| Uncooked pizzas | Food cupboard → pineapple |
| Whole pineapple | Knife + pineapple in inventory |
| Pineapple chunks + uncooked pizza | Clay oven |
| Pineapple chunks + plain pizza | Plain pizza + chunks in inventory |

## Setup

Bring 13 bowls, 1 knife, and 13 free inventory slots to the Hosidius Mess Hall.
